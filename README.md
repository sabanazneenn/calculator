**Calculator App:**

This is a small project to demonstrate a calculator application with CI/CD automation.

**Features:**

Basic operations: add, subtract, multiply, divide

Built with Flask

Runs inside Docker

Automated builds and deployments using Jenkins

**CI/CD FLOW**

Push code to GitHub

Jenkins job triggers automatically via webhook

Docker image rebuilt and redeployed

App available instantly on port 6060

This project is a simple but clear example of how CI/CD pipelines work with GitHub, Jenkins, and Docker.
